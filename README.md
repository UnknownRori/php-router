# PHP-Router

A simple dynamic route matcher for php

## 🛠️ Development

```bash
# clone the repository
> git clone https://github.com/UnknownRori/php-routes

# enter directory
> cd php-routes

# install dependency
> composer install

# do some test
> ./vendor/phpunit/phpunit/phpunit
```
